# MODBUS Sample Code for ML62Q1000 Series

This sample code demonstrates the usage of MODBUS communication for the ML62Q1000 series of Lapis processors, specifically the ML62Q1722 MCU.

# Overview

The provided code implements a MODBUS using the FreeMODBUS library. It showcases basic setup of MODBUS communication on the ML62Q1722 MCU.

## Pin Configuration

The sample code uses the following pin configuration for RS485 UART communication:

- P20: Transmit (TX) pin for
- P17: Receive (RX) pin
- P70: Enable (EN) pin
